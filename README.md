# Projeto_HTML
Projeto de HTML como ferramenta de auxílio para cálculos contábeis destinados a pessoas empregadas regidas pelo regime de contratação CLT.  
Em fase de aprimoramento, contendo ferramentas como:
1 - Cálculo de recisão
2 - Cálculo de férias
3 - Ferramentas de PJ
4 - Artigos informativos
